About Air String Pro

Air String Pro represents the future of musical expression, born from the vision to free musicians from the constraints of physical instruments. Powered by cutting-edge OpenCV technology for precise hand tracking, Pygame MIDI for responsive sound generation, and Tkinter for an intuitive interface, this instrument opens new possibilities for creative expression.

Whether you're a professional musician looking to expand your sonic palette or a beginner exploring the world of music production, Air String Pro provides an accessible yet powerful platform for your musical journey.

Specifications to Run This Project:

To ensure smooth performance, you will need:

  • A computer with a webcam or a smartphone Camera
  • Python 3.10 or later
  • OpenCV for hand tracking
  • cvzone for simplified computer vision functions
  • pygame.midi for MIDI sound generation

Technologies Used:

This project is built using OpenCV for real-time hand tracking, cvzone for gesture detection, and pygame.midi for playing MIDI notes. These technologies work together to create an intuitive and interactive air instrument experience.
